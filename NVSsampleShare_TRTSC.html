Overview of Speech

This page contains three main sections:

  • Original NVS: contains the original NVS samples.
  • Time reversed NVS: contains the time-reversed NVS samples of original speech.
  • No processed NVS: contains the original NVS samples with no prossed.
  • Stretched NVS: contains the temporal stretched NVS samples.
  • Compressed NVS: contains the temporal compressed NVS samples.
  • Stretched and stretched NVS: contains the temporal stretched and compressed NVS samples.
